    ---------- Important Notes ----------

    * In HTML for Un-list items, by default round disc has come before the list and the Tag which is generally used for this is "ul" tag.. and when we have to remove that disc style we have to write under the Punctuation called "type=none" under "< >" this Punctuation..

    * In CSS for the same style we have to write the style called "style=list-style-type:none" under the same punctuation..so that the default style of Round disc of "ul" tag has disappear..

    * The same Thing in bootstrap we have to write the Attribute called :- class="list-unstyled" under the same Punctuation so, that the styles will apply..
