  5. Responsive Text Alignment, are Alignment which respond According to the size of the screen, the screen were divided into 4 parts..

    1. Mobile Screen Type "576px" (sm)
    2. Tab. Screen Type "776" (md)
    3. Laptop Screen Type "992px" (lg)
    4. Extra-large / Projector Type "1200px" (xl)

    When we put Responsive Text-Alignment for the given screen type then whatever you give attribute for that screen type that styles only apply for that particular screen and remaining screen will be default type..
