Spacing in Bootstrap4

  1. Bootstrap Margin , Padding(0,1,2,3,4,5,auto)

Bootstrap Margin and Padding are the Same Content Style of CSS which we had already read in the previous class in this concept of Bootstrap we have to learn about how we can use the same thing in Bootstrap with a shortcut key that apply the same effect for the content..


Some Shortcuts are available :-

  • Padding - (0,1,2,3,4,5,auto)

  • Padding :-
    Padding-left(pl) ==> towards X-axis(px)
    Padding-right(pr)
    Padding-top(pt) ==> towards Y-axis(py)
    Padding-bottom(pb)
    (p) ==> For all side
  • Margin :-
    Margin-left(ml) ==> towards X-axis(mx)
    Margin-right(mr)
    Margin-top(mt) ==> towards Y-axis(my)
    Margin-bottom(mb)
    (m) ==> For all side
