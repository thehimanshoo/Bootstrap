  1. Float, as we have read about it in "CSS" that it is a property which used to float some text to Left or Right.. Actually by default it always float left but when we give an Attribute called "float-left / float-Right " it sticks to that position.. In "CSS" To apply this style, the Attribute used called :- display: float;

    But, in Bootstrap if you want to use this property then you will have to give the different Attribute for the same style and the "Attribute" is :-

    • float-left

    • float-right

  2. Responsive Float


    3. Responsive Float refers to the style of float that responses according ot the screen size.. As we already knows and read in "CSS" Float Box Concept, but that were non-responsive float but this concept of "Bootstrap" Provides much better style as compared to "CSS" and also if you write a style for a (sm) type screen then it only apply to that special one and remaining screen style will set to the default until you don't write a style for that other one..

Clearfix is a property of Bootstrap it is generally used when we have to "Float" Something to Left-Side and "Float" Something to Right-Side, then the default nature of "Float" is the middle side between both the Floated thing and covers that space.. By this Clearfix-Property of Bootstrap the covered space which was occupy by the Paragraph Content will get clear..
